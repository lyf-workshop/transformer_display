# 象限分类神经网络

用一个简单的全连接神经网络，判断平面直角坐标系中的点 **(x, y)** 属于哪个象限。

## 任务描述

| 类别编号 | 含义 | 条件 |
|---------|------|------|
| 0 | 第一象限 | x > 0, y > 0 |
| 1 | 第二象限 | x < 0, y > 0 |
| 2 | 第三象限 | x < 0, y < 0 |
| 3 | 第四象限 | x > 0, y < 0 |
| 4 | 坐标轴上 | x = 0 或 y = 0 |

> 将 `NUM_CLASSES = 4` 可去掉坐标轴类别，变为纯 4 分类。

## 网络结构

```
输入层   →  2 个神经元 (x, y)
隐藏层1  →  16 个神经元 + ReLU
隐藏层2  →  8  个神经元 + ReLU
输出层   →  5  个神经元 + Softmax（CrossEntropyLoss 内置）
```

总参数量：**357**（极简，纯靠坐标正负关系即可完美分类）

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 训练 + 评估 + 可视化
python quadrant_classifier.py
```

运行后会：
1. 打印每 20 个 epoch 的 loss 和测试集准确率
2. 输出每个类别的单独准确率
3. 演示 5 个测试点的预测结果及各类概率
4. 保存三联可视化图 `quadrant_result.png`（损失曲线 / 真实标签散点图 / 决策边界）

## 可调超参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `N_QUADRANT` | 2000 | 每个象限的训练样本数 |
| `N_AXIS` | 400 | 坐标轴类别的样本数 |
| `COORD_RANGE` | 5.0 | 坐标取值范围 ±5 |
| `HIDDEN1` | 16 | 第一隐藏层宽度 |
| `HIDDEN2` | 8 | 第二隐藏层宽度 |
| `NUM_CLASSES` | 5 | 分类数（4 或 5） |
| `EPOCHS` | 200 | 训练轮数 |
| `LR` | 1e-3 | Adam 学习率 |

## 预期结果

由于象限判断本质上只需判断 x、y 的正负，网络收敛极快：

- 约 **20 epoch** 后测试准确率达到 **99%+**
- 200 epoch 结束后通常可达到 **100%**（四象限类别）
- 坐标轴类别因样本少且边界尖锐，准确率略低但仍 >95%

## 单点预测

训练完成后可直接调用 `predict_point(x, y)`：

```python
result = predict_point(3.0, -2.0)
# {'坐标': (3.0, -2.0), '预测象限': '第四象限', '各类概率': {...}}
```
